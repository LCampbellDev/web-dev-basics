Web Development Basics

Tips and free resources you can use to build your skills

Why learn web development?

Why not! There are so many transferrable skills to be learnt from coding, from problem solving to resilience, to keeping brain working and healthy by forming new neural pathways, to actually becoming a web developer or software engineer. So, give it a go and see if you enjoy it.

"Coding is a great hobby, I love how I can dom it anytime, on my own and get that dopamine hit from building something tangible."

What exactly is web development?

Web development foundations most commonly involve:

  • In a nutshell, building websites.
  • HTML- the strucure and layout of the page (remember early 90s websites? these were built with html only).
  • CSS - This adds extra styling to make websites more visually appealing.
  • JavaScript for adding fancy interactive things like animations, dark mode, and visual changes that happen when you hover your mouse or click in certain areas of the page.
    • >

There are other ways to build websites, however for a beginner this is the simplest way to get started.

The coding knowledge and skills you develop in HTML, CSS and JavaScript for web development can be used specific to that role, or be a good foundation to build on with further programming and software engineering skills.

"I've met and made friends with people I otherwise wouldn't have got to know through attending local coding meetups. This has also been a great way to get feedback and mentoring and to find open source projects to get involved in."
